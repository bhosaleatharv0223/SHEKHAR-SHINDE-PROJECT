## Section Breakdown

### 1. Hero Section
- **Height**: Auto (py-20)
- **Background**: #f8fafc
- **Animation**: Fade up (0.5s)
- **Elements**: 
  - Heading (clamp 2-3rem)
  - Decorative line (48px × 3px)
  - Subheading

### 2. Contact Method Cards
- **Layout**: 3-column grid (1-column mobile)
- **Gap**: 1.5rem
- **Max Width**: 1000px
- **Animation**: Slide up + Gentle float
- **Cards**: Call, WhatsApp, Email

### 3. Combined Office + Branches
- **Layout**: 2-column grid (1-column mobile)
- **Max Width**: 1000px
- **Border**: 1.5px solid #e2e8f0
- **Animation**: Slide from left/right
- **Divider**: 1px vertical line

### 4. Business Hours
- **Width**: 60% (max 600px)
- **Position**: Centered
- **Animation**: Scale in + Gentle float
- **Rows**: 3 (Mon-Fri, Sat, Sun)

### 5. Contact Form
- **Max Width**: 720px
- **Position**: Centered
- **Padding**: 2.5rem 3rem
- **Animation**: Slide up + Field stagger
- **Fields**: 6 inputs + 1 textarea

### 6. FAQs
- **Max Width**: 800px
- **Position**: Centered
- **Items**: 5 accordion items
- **Animation**: Stagger up + Chevron rotate

### 7. Instant Help Banner
- **Max Width**: 900px
- **Position**: Centered
- **Background**: Linear gradient blue
- **Buttons**: 2 (Call + WhatsApp)
- **Animation**: Scale in

## Responsive Breakpoints

```css
/* Mobile First */
Default: Single column, full width

/* Tablet (768px+) */
- Contact cards: 2 columns
- Office/Branches: Still stacked
- Form: Wider padding

/* Desktop (1024px+) */
- Contact cards: 3 columns
- Office/Branches: 2 columns side-by-side
- All sections: Max-width constraints active
```


## Animation Timeline

```
0.0s  → Hero heading fades up
0.2s  → Subheading fades up
0.3s  → Decorative line slides in
0.4s  → Contact cards start appearing (staggered 0.1s each)
0.6s  → Office section slides from left
0.7s  → Branches section slides from right
0.8s  → Branch rows stagger in
1.0s  → Business hours scales in
1.2s  → Form card slides up
1.3s  → Form fields stagger in
1.5s  → FAQ heading appears
1.6s  → FAQ items stagger in
1.8s  → Help banner scales in

Continuous:
- Contact cards float (4s infinite)
- Business hours floats (4s infinite, delay 0.5s)
```

## Interaction States

### Hover Effects
- **Contact Cards**: translateY(-6px), border color change, shadow increase
- **Logo Circles**: scale(1.06)
- **Buttons**: translateY(-2px), shadow increase
- **Branch Rows**: background tint, border color
- **FAQ Items**: background tint

### Focus States
- **Form Fields**: 
  - Border: #2563eb
  - Shadow: 0 0 0 3px rgba(37,99,235,0.10)
  - Background: white

### Active States
- **Buttons**: scale(0.96-0.98)
- **Accordion**: Chevron rotates 180deg, content expands

## Spacing System

```
Section Margins:
- Between sections: 2-3rem
- Hero to cards: 3rem
- Cards to combined: 2rem
- Combined to hours: 1.5rem
- Hours to form: 3rem
- Form to FAQs: 3rem
- FAQs to banner: 3rem

Card Padding:
- Contact cards: 2rem 1.5rem

- Business hours: 1.75rem 2rem
- Form card: 2.5rem 3rem

Element Gaps:
- Card grid: 1.5rem
- Form fields: 1.25rem
- FAQ items: 0.75rem
- Branch rows: 0.5rem
```
